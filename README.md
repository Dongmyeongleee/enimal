# 🐣 기획배경

✔ 환경 문제, 서식지 부족 등으로 멸종위기 종이 빠르게 늘어나고 있습니다. 

✔ 집중적인 보전 활동을 한다면 멸종위기 종의 서식지 환경이 개선되고, 개체 수가 증가할 수 있지만 돈과 관심이 필요합니다. 

✔ 이런 생각을 토대로 저희는 멸종위기 종에 대한 관심을 유발하고, 기부를 독려할 수 있는 NFT 서비스 Enimal (Endangered + Animal)을 만들게 되었습니다.
